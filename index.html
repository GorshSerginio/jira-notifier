Notifier for JIRA

JIRA desktop client for tracking changes to the issues

Notifier for JIRA is a Desktop JIRA-client for tracking changes to the issues in which you are interested. For now It's available ony for Windows.

Installation

  • Download the latest version
  • Launch the installer and follow the instructions
